Materiales del Curso 📚

Contenido del curso de Programación Web — HTML5, CSS3, enlaces, formularios y más.

🌐 Unidad 1 — HTML5

Estructura básica

  • DOCTYPE y etiqueta html
  • head, body, meta
  • Semántica: header, nav, main, footer
  • section, article, aside

Etiquetas de contenido

  • Encabezados h1–h6
  • Párrafos, span, div
  • Listas: ul, ol, dl
  • Tablas: table, tr, td, th

Multimedia

  • Imágenes: img con src, alt
  • Video: video con controls
  • Audio: audio con src
  • iframe para contenido externo

Formularios HTML5

  • form, input (text, email, date...)
  • textarea, select, option
  • button, label, fieldset
  • Validaciones: required, pattern

🎨 Unidad 2 — CSS3

Selectores y especificidad

  • Selectores de elemento, clase, id
  • Pseudo-clases: :hover, :focus
  • Pseudo-elementos: ::before, ::after
  • Especificidad y cascada

Box Model

  • margin, padding, border
  • box-sizing: border-box
  • width, height, max-width
  • display: block, inline, inline-block

Flexbox

  • display: flex
  • flex-direction, flex-wrap
  • justify-content, align-items
  • flex: 1, gap

Responsive Design

  • Media queries: @media
  • Unidades: %, vw, vh, rem, em
  • CSS Grid básico
  • Viewport meta tag

🔗 Unidad 3 — Enlaces y Navegación

Tipos de enlaces

  • Enlaces relativos: href="pagina.html"
  • Enlaces absolutos: href="https://..."
  • Anclas: href="#seccion"
  • target="_blank" para nueva pestaña

Atributos importantes

  • href — destino del enlace
  • src — fuente de imagen/media
  • alt — texto alternativo
  • class, id — selectores CSS/JS
Ejemplo de enlace absoluto: <a href="https://developer.mozilla.org" target="_blank">MDN Docs</a>
